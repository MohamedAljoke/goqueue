# Video 03 – Making Jobs Flexible with Handlers

## Branch

video-03-handler-function

## Goal

Allow jobs to execute different behaviors.

## What We Build

- `HandlerFunc` type
- Pass handler into `Process()`

## Concepts

- Functions as first-class citizens
- Passing behavior instead of conditionals
- Cleaner job execution

## Example Use Cases

- Email job
- SMS job
- Report generation job

## Teaching Notes

Avoid buzzwords like “dependency injection”.
Explain it as “plugging behavior in”.
